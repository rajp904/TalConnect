# 💼 TalConnect – AI Recruitment Platform

A full-stack MERN application that connects job seekers with recruiters using AI-powered resume analysis.

## Features
- Authentication (JWT)
- Job listing & application system
- AI-based resume scoring
- Email notifications
- Responsive UI

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- LangChain (AI)
